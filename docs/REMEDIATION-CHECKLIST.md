## ملاحظات تشغيلية مفتوحة

- NextAuth `5.0.0-beta.31` يعلن peer اختياريًا لـ Nodemailer 7، بينما المشروع يستخدم Nodemailer 9.0.3 الآمن. ضُبط npm على `legacy-peer-deps` إلى أن يحدّث NextAuth نطاق الـ peer.
- التخزين المشترك لمحدد المعدل يحتاج تهيئة Redis/Upstash أو خدمة مماثلة؛ التنفيذ الحالي آمن لنسخة تطبيق واحدة فقط.
